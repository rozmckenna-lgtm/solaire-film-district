# Solaire Film District

Site institutionnel du Solaire Film District.

**URL** : [https://solairefilmdistrict.fr](https://solairefilmdistrict.fr)

## Stack technique

- HTML5 semantique
- CSS3 (custom properties, Grid, Flexbox)
- Formulaire : [Formspree](https://formspree.io)
- Hebergement : GitHub Pages

## Structure

Voir [DOCUMENTATION.md](DOCUMENTATION.md) pour la documentation complete.

## Developpement local

Ouvrir `index.html` dans un navigateur, ou utiliser un serveur local :

```bash
python3 -m http.server 8000
```

Puis aller sur `http://localhost:8000`.
